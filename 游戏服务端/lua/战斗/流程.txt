战场
    战斗开始->创建死循环协程{
        {
            rpc -> 客户端创建
            打开菜单 -> 判断BUFF等
            回合开始 -> BUFF
            客户端 播放回合开始
            回合演算 -> 速度排序 -> 判断合击 -> 演算
            rpc -> 播放回合演算
            回合结束
            客户端 播放回合结束
        }loop
        战斗结束
    }
